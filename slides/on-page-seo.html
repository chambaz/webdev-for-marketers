On-page SEO

What is On-page Search Engine Optimization (SEO)?

On-page SEO is the practice of optimizing individual web pages in order to rank higher and earn more relevant traffic in search engines. On-page refers to both the content and HTML source code of a page that can be optimized - Moz.com

HTML denotes meaning to text documents for browsers to read and understand.

On-page SEO is the process of providing further meaning for Search Engines to understand.

The Head and The Body

A webpage has two main parent elements, the head contains information about the page, and the body contains the actual visible content.

The Head Element

<head>

The Head Element

The head element is not visible to users and acts as a place for developers to provide browsers with resources and information regarding the webpage itself, like the page title and description.

The Title Element

<title>

The Title Element

The title element denotes the title of the page and is one of the most important elements in search engine optimization.

The Meta Element

<meta>

The Meta Element

The meta element can be used within the head of a document to specify information about the page itself

Remember HTML attributes?

<element attribute="value">

The Meta Element

The meta element has two attributes, name and content, and there are a number of different uses distinquished by their 'name', including the most common the meta 'description'

The Head of a Webpage

<head>

    <title> This is the title of my page </title>

    <meta name="description" content="This is my page description" />

</head>

HTML attributes?

There are several HTML attributes that can be advantageous to SEO if used smartly. DO NOT try and trick Google, it won't work.

The ALT Attribute

Image elements have an optional 'alt' attribute, for providing an alternative text based description of the image. The intention is for this to describe the image in the case that an image does not load correctly, or for users with screen readers or other accesiblity issues.

The ALT Attribute

<img
       src="image-of-ds.jpg"
       alt="Digital Surgeons digital marketing agency based in New Haven">

The Title Attribute

Image and Anchor (link) elements have an optional 'title' attribute, for providing a text based description to a user in the form of a tool tip before they click to navigate.

The Title Attribute

<img
       src="image-of-ds.jpg"
       title="Digital Surgeons digital marketing agency based in New Haven">

<a
       href="https://www.digitalsurgeons.com"
       title="Digital Surgeons digital marketing agency based in New Haven">
</a>
