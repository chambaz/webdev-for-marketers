Cascading Style Sheets

Why cascading?

Why cascading?

There are several levels of priority in CSS, styles with higher priority will overwrite those proceeding, this is know as cascading.

Seaparating Concerns

Seaparating Concerns

Separating our markup and stylesheets allow us to have more fine-grained control over styling.

<head>

    <title> This is the title of my page </title>

    <style>
        // styles go here
    </style>

</head>

<head>

    <title> This is the title of my page </title>

    <link href="styles.css" />

</head>

Selectors

Selectors

Selectors are patterns used to select the element(s) you want to style.

Selectors

<h3 class="my-heading">Hello world</h3>

}
