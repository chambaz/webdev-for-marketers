What is HTML

HTML is a programming language

HyperText Markup Language

Semantically mark up text documents for browsers to read, understand, and render.

HyperText Markup Language

Create structured documents by denoting semantics for text

Semantics?

Semantics (from Ancient Greek: σημαντικός sēmantikos, "significant") is primarily the linguistic, and also philosophical, study of meaning in language.

HTML denotes meaning for pieces of content within a document.
